# TemplateFiles
A set of template files for Ubuntu user to create file easily.

<br/>

## Supported Template Files
```
/Templates
    /Empty Text File.txt
    /Graphics
    /Program Files
    /Script Files
    /Office
    /Others
```

<br/>

## Installation Guide
1. Download this directory.
1. Extract the `Templates` directory into your `Templates` directory inside User Home.
